# Portfolio Website

A modern, responsive portfolio website built with HTML, CSS, and JavaScript.

## Features

- 🎨 Dark modern design with purple/blue accents
- 📱 Fully responsive layout
- ⚡ Smooth scroll navigation
- ✨ Animated elements and transitions
- 🌙 Mobile-friendly hamburger menu

[View Live Site](https://your-username.github.io/portfolio/)

## Deployment

### GitHub Pages

1. Push this repository to GitHub
2. Go to Settings > Pages
3. Select the `main` branch and save
4. Your site will be live at `https://your-username.github.io/portfolio/`

### Netlify

1. Push to GitHub first
2. Go to [Netlify](https://netlify.com)
3. Click "Add new site" > "Import an existing project"
4. Connect your GitHub account and select this repo
5. Deploy settings:
   - Build command: (leave empty)
   - Publish directory: `/` (root)
6. Click "Deploy site"

## Customization

Edit `index.html` to update:
- Your name and title
- Project details
- Contact information
- Social media links

Edit `styles.css` to change:
- Color scheme (see CSS variables in `:root`)
- Fonts
- Spacing and layout
